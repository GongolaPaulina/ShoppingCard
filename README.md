##  ShoppingCard is a simple pattern that behind items in a shopping card.

Each row represents product. 
There are the quantity of the product and there are buttons to change that quantity.
- Delete button remove item from the list.
- Reset button reset the quantity of all the items to zero.

There are badge on the navigation bar which is like e-commerce application. As you add items in the shopping card it shows the total number of item in the shopping cart but in this application it behaves a little bit differently: it only shows the number of counters with value greater than zero.
